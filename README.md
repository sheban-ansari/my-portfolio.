# Responsive Portfolio Website (Flexbox & Grid)

A fully responsive portfolio website built using HTML5 and CSS3.  
This project demonstrates the use of **CSS Flexbox** for layout alignment and **CSS Grid** for organizing content sections.

---

##  Project Objective

The goal of this project was to:

- Build a responsive webpage layout
- Use Flexbox for page structure and alignment
- Use CSS Grid for organizing project sections
- Ensure responsiveness across desktop, tablet, and mobile devices

---

[Click Here to View Project](https://my-portfolio-zzpm.vercel.app/)

---

##  Technologies Used

- HTML5
- CSS3
- CSS Flexbox
- CSS Grid
- Media Queries

---

##  Features

- Responsive navigation bar using Flexbox
- Hero section with modern gradient design
- Sidebar layout using Flexbox
- Project section organized using CSS Grid
- Hover animations on project cards
- Fully responsive layout (Desktop / Tablet / Mobile)
- Smooth scrolling navigation
- Contact section with email, GitHub, and LinkedIn links

---

##  Responsive Design

The website adapts to different screen sizes using media queries:

- Desktop: Multi-column layout
- Tablet: Stacked layout with single-column project grid
- Mobile: Optimized padding and centered navigation

Breakpoints used:
- 992px
- 600px

---

##  Layout Implementation

###  Flexbox Used For:
- Header alignment
- Navigation menu
- Sidebar and main content layout
- Project links alignment

###  CSS Grid Used For:
- Organizing project cards
- Creating a responsive grid layout
